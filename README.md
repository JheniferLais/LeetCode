# LeetCode

Este repositório contém soluções para diversos desafios do LeetCode, focados em algoritmos e estruturas de dados. 
O principal objetivo é praticar para entrevistas técnicas e aprimorar a lógica de programação.

As soluções estão escritas em Java (algoritmos) e C++ (estruturas de dados).

## 🎯 Objetivos
- Praticar problemas de lógica e algoritmos.
- Reforçar os conceitos de estruturas de dados.
- Preparar para entrevistas e testes técnicos.
